## Descrição
Como **Gerente de Unidade**, preciso visualizar o histórico completo de cada cliente (comandas, orçamentos, frequência, ticket médio) e registrar notas de atendimento, para manter um relacionamento próximo e identificar oportunidades.

---

## Critérios de Aceite

- [x] **AC1:** Página `/unit/[unitId]/clients/[clientId]` com detalhe do cliente
- [x] **AC2:** Estatísticas: total de pedidos aprovados, total gasto, ticket médio
- [x] **AC3:** Nome do cliente na listagem é um link para o detalhe (histórico de orçamentos aprovados via stats)
- [x] **AC4:** Notas de atendimento: criar nota com categoria (visita/ligação/email/outro) e texto
- [x] **AC5:** Lista das últimas 10 notas de atendimento com data e autor

---

## Escopo
**IN:** Detalhe de cliente, estatísticas, notas de atendimento
**OUT:** NPS automático (Wave 3), automações de e-mail (Wave 4)
