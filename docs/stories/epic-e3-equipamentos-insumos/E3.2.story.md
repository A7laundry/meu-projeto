## Descrição
Como **Operador ou Gerente**, preciso registrar ocorrências e o uso diário das máquinas no diário de bordo, para ter rastreabilidade de quem usou cada equipamento, quantos ciclos foram realizados e quais problemas ocorreram.

---

## Critérios de Aceite

- [x] **AC1:** Página de detalhe do equipamento em `/unit/[unitId]/equipment/[equipmentId]`
- [x] **AC2:** Formulário para registrar entrada no diário: tipo (use, maintenance, incident, repair_completed), observações, ciclos realizados (se type=use)
- [x] **AC3:** Histórico de entradas do diário em ordem cronológica reversa (mais recente primeiro)
- [x] **AC4:** Status do equipamento atualiza automaticamente: quando type=maintenance → status=maintenance; quando type=repair_completed → status=active
- [x] **AC5:** Contador de ciclos acumulados visível na página de detalhe (total de ciclos registrados)
- [x] **AC6:** Listagem de equipamentos na página principal mostra link para a página de detalhe
