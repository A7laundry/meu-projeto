# Story E4.3 — Interface do Motorista
**Epic:** E4 — Logística
**Status:** Ready for Review
**Criado por:** @sm (River) | 2026-02-19
**Complexidade:** M (5 pontos)

---

## Descrição
Como **Motorista**, preciso de uma interface mobile-friendly para ver meu romaneio do dia e marcar cada parada como visitada ou pulada, para que o gerente acompanhe o progresso da rota em tempo real.

---

## Critérios de Aceite

- [x] **AC1:** Rota `/driver/[manifestId]` mostra o romaneio do motorista (sem sidebar, layout mobile via grupo `(driver)`)
- [x] **AC2:** Lista de paradas com endereço, botão "Visitei" e "Pular"
- [x] **AC3:** Progresso da rota exibido (X de N paradas visitadas com barra de progresso)
- [x] **AC4:** Botão "Concluir Rota" muda status para `completed` quando todas as paradas foram tratadas
- [x] **AC5:** Link "Ver (motorista)" em cada card de romaneio na página de gestão abre a view do motorista em nova aba

---

## Escopo
**IN:** Interface mobile de execução de rota, atualização de paradas
**OUT:** GPS/navegação (Wave 3), assinatura do cliente (Wave 3), notificações push (Wave 4)
