**Epic:** EPIC-MVP

**Estimativa:** 1 dia

## Objetivo

Notificar o cliente automaticamente quando o pedido muda de status. Email ja funciona via Resend. Adicionar WhatsApp (link) e melhorar templates.

## Acceptance Criteria

### AC1: Email Automatico por Mudanca de Status
- [ ] Disparar email nos status: sorting, ready, shipped, delivered
- [ ] Template HTML limpo com: logo, numero do pedido, status atual, proxima etapa
- [ ] Usar Resend (ja configurado)
- [ ] Somente se cliente tem email cadastrado

### AC2: WhatsApp Quick Link
- [ ] Gerar link wa.me com mensagem pre-formatada por status
- [ ] Mostrar botao "Notificar via WhatsApp" no detalhe do pedido
- [ ] Mensagem: "Ola {nome}! Sua comanda {numero} esta em {status}. Previsao: {data}"
- [ ] Somente se cliente tem telefone cadastrado

### AC3: Portal do Cliente - Realtime
- [ ] Pagina /client/orders ja existe
- [ ] Adicionar Supabase Realtime subscription para order_events
- [ ] Toast notification quando status muda
- [ ] Timeline visual de eventos atualizada em tempo real
